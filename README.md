# ifLink Quest｜共創DX人材育成OS IF-THENカード実装版

GitHub Pages公開用の静的Webアプリです。

## 公開手順
1. このフォルダの中身をGitHubリポジトリ直下へアップロード
2. Settings → Pages
3. Deploy from a branch / main / root を選択
4. 公開URLへアクセス

## 実装機能
- IFカード、THENカード、ASSETカードの選択・組み合わせ
- 課題カード選択
- IF-THENレシピ生成
- PoC企画書・研修ログ生成
- 独自カード追加CMS
- localStorage保存
